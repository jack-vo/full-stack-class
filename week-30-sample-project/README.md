# Project Documentation
## Main Ideas
- The project is a simple blog post application that is similar to WordPress.

## Target Audience
- Blog post editor
- Blog post reader

## Features
### 1. Blog Post Editor
- Editor can sign up using:
  - Email / Password
  - Google account
- After signing in, editor can:
  - Create a new blog post or edit an existing blog post
    - Blog post contains:
      - Title
      - Content (Rich Text Editor)
      - Images
      - Videos
      - Tags
  - Delete a blog post
  - View all blog posts

### 2. Blog Post Readers
- Readers can:
  - View all blog posts
  - View a single blog post
  - Leave a comment on a blog post after signing in (optional)
